## Шаг 7б. Обработка нестандартных барьеров (AI-элемент)

Открытое поле "мой страх своими словами" обрабатывается так:

1. Система определяет категорию страха по ключевым словам
2. Показывает шаблонный обход для этой категории
3. Claude добавляет 2-3 строки персонального комментария —
   с учётом конкретной формулировки И цвета профиля человека

**Важно:** ответ знает цвет профиля. Один и тот же страх у Архитектора
и у Творца требует разного обхода.

Технически: агент уровня 2 с контекстом профиля + банком барьеров.
Реализуется через Claude API прямо на странице результата.

## Шаг 10. Техническая реализация

### Варианты платформы для квиза
| Платформа | Плюсы | Минусы | Рекомендация |
|---|---|---|---|
| Tally | Бесплатно, красиво, условная логика | Нет встроенного алгоритма расчёта | Для MVP если результат отдаётся вручную или через Zapier |
| Typeform | Красивый UX, логика ветвления | Платно от $25/мес | Если важен внешний вид на старте |
| Lovable (собственная) | Полный контроль, алгоритм внутри, Claude API | Нужно собрать | Оптимально: алгоритм + страница результата + AI-агент в одном |
| Notion + расчёт вручную | Быстро, ноль кода | Не масштабируется | Только для теста концепции |

**Рекомендация для MVP:** Tally (квиз) → Google Sheets (расчёт по алгоритму) → Notion/лендинг (страница результата). Потом — переехать на Lovable с Claude API.

### Платёжная система
- **ЮКасса** — если аудитория РФ/СНГ
- **Stripe** — международная, простой вход
- **Gumroad** — цифровые продукты, встроена оплата, комиссия 10%
- **Telegram Stars** — если результат выдаётся через бота
