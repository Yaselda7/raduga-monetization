## Алгоритм расчёта профиля

### Шаг 1: Вычислить оси

| Ось | Формула | Интерпретация |
|---|---|---|
| **Температура денег** | Тепло = Q5; Холод = (Q6 + Q13) / 2 | Тепло > Холод → тёплая группа |
| **Создание ↔ Взаимодействие** | Взаимодействие = (Q4 + Q9) / 2; Создание = Q10 | |
| **Структура ↔ Гибкость** | Структура = Q2; Гибкость = (5 − Q2) | |
| **Глубина ↔ Скорость** | Глубина = (Q1 + Q12) / 2; Скорость = Q11 | |
| **Автономия ↔ Контакт** | Автономия = (Q3 + Q10) / 2; Контакт = (Q4 + Q9) / 2 | |
| **Проявление ↔ Избегание** | Проявление = Q8; Избегание = (Q7 + Q14) / 2 | |
| **Миссия ↔ Прагматика** | Q15 | Высокий = миссионерский, низкий = прагматичный |

### Шаг 2: Определить группу по температуре

**Если Тепло > Холод → тёплая группа:** Спринтер / Творец / Просветитель / Наставник

**Если Холод > Тепло → холодная группа:** Архитектор / Аналитик / Мастер

---

### Шаг 3: Определить профиль внутри группы

#### Тёплая группа

| Профиль | Ключевые признаки |
|---|---|
| 🔴 **Спринтер** | Скорость высокая (Q11 ≥ 4) + Структура умеренная (Q2 ≥ 3) + Проявление высокое (Q8 ≥ 4) |
| 🟠 **Творец** | Скорость высокая (Q11 ≥ 4) + Гибкость высокая (Q2 ≤ 2) + Проявление высокое (Q8 ≥ 4) |
| 🟢 **Просветитель** | Глубина высокая (Q1+Q12 ≥ 4) + Взаимодействие высокое (Q4+Q9 ≥ 4) + Скорость низкая (Q11 ≤ 2) |
| 🔵 **Наставник** | Взаимодействие высокое (Q4+Q9 ≥ 4) + Скорость средняя (Q11 2–3) + Структура умеренная (Q2 ≥ 3) |

**Правила при совпадении:**
- Спринтер vs Творец → Q2: если ≥ 3 = Спринтер, если ≤ 2 = Творец
- Просветитель vs Наставник → Q1+Q12: если Глубина > Скорость = Просветитель

#### Холодная группа

| Профиль | Ключевые признаки |
|---|---|
| 🟣 **Архитектор** | Избегание высокое (Q7+Q14 ≥ 4) + Глубина высокая + Миссия высокая (Q15 ≥ 4) |
| 💙 **Аналитик** | Структура высокая (Q2 ≥ 4) + Глубина высокая + Миссия низкая (Q15 ≤ 2) |
| 🟡 **Мастер** | Глубина высокая (Q12 ≥ 4) + Избегание умеренное + Взаимодействие низкое + Миссия нейтральная |

**Правила при совпадении:**
- Архитектор vs Аналитик → Q15: если ≥ 4 = Архитектор, если ≤ 2 = Аналитик, если 3 = оба почти равны, смотри Q7+Q14
- Аналитик vs Мастер → Q2: если ≥ 4 = Аналитик, если ≤ 2 = Мастер
