# Продукты вайбкодинга — матрица для Радуги монетизации

## Уровни продуктов

### Уровень 1 — Один агент, локальная память под функцию
Техника создания одна — меняется предметная область.
Быстрый старт, первые деньги, проверка ниши.

Примеры:
- Лендинг для специалиста / продукта
- Калькулятор стоимости услуг
- Калькулятор для записи / бронирования
- Бот с уведомлениями (клиентам и мастеру)
- Telegram-бот для записи и FAQ
- Парсер для поиска клиентов
- Генератор документов (договоры, КП, NDA)
- Квиз с результатом
- Форма с автоматической обработкой
- Цифровые продукты (шаблоны, чек-листы, гайды)
- Портфолио-сайт

### Уровень 2 — Пара агентов с общей памятью
Примеры:
- CRM для управления лидами
- Система онлайн-записи с напоминаниями
- Бот клиентского сервиса (квалификация + передача)
- Автоматическая email-воронка
- Персональный ИИ-ассистент для бизнеса
- Агент мониторинга трендов
- Агент анализа звонков
- Система онбординга клиентов
- Дашборд аналитики (продажи, финансы, реклама)
- Платформа цифровых товаров с автовыдачей
- Трекер привычек / wellness-продукт
- HR-бот для найма

### Уровень 3 — Мультиагентная система с оркестратором
Шаблонов нет — каждый проект уникален.
Конкурентное преимущество, более высокий чек.

Примеры:
- Платформа онлайн-обучения с личным кабинетом и трекером прогресса
- Платформа платного клуба по подписке
- Платформа для проведения марафонов и мастермайндов
- Микро-SaaS по подписке (нишевый инструмент)
- Мультиагентная система для бизнес-процессов
- Платформа с дашбордом проектов (Supabase + агенты)
- Система управления командой с ИИ
- Умная CRM с предсказанием продаж
- Платформа для онлайн-консультаций
- Система геймификации обучения
- Платформа для нетворкинга с ИИ-подбором

### Уровень 4 — Агент-создатель (в перспективе)
Агент, который сам строит других агентов.
Требует специализированных платформ (изучается).
Следующий поток курса работает с этим уровнем.

## Матрица: профиль → естественные продукты

### 🟣 Архитектор (Фиолетовый)
**Стремится к:** уровень 3-4
**Риск:** застрять в разработке, не выйти на рынок

Естественные продукты:
- Микро-SaaS по подписке
- Платформы с агентами под конкретную нишу
- Мультиагентные системы для бизнес-процессов
- Платформа с дашбордом и аналитикой

Формат продажи: разовая оплата, продукт работает без автора

Антипаттерн: курсы, воркшопы, живое сопровождение

---

### 💙 Аналитик (Синий)
**Стремится к:** уровень 3-4
**Риск:** идеальная система, которую никто не купит — потому что не объяснена

Естественные продукты:
- Дашборды аналитики (продажи, реклама, финансы)
- CRM и системы управления
- Методологии и агентские системы
- Аудиты и исследования с ИИ
- Системы управления командой

Формат продажи: через кейсы и рекомендации, высокий чек

Антипаттерн: агрессивный маркетинг, массовые продажи

---

### 🔵 Наставник (Голубой)
**Стремится к:** уровень 2-3, выпускает быстро
**Риск:** распыление, много начатого

Естественные продукты:
- Обучающие платформы и курсы
- Боты-наставники с базой знаний
- Системы онбординга
- Платформы для сообществ и клубов
- Мультиагентные обучающие системы

Формат продажи: через контент и живое присутствие, подписка или разовая оплата

Антипаттерн: монотонный технический продукт без живого контакта

---

### 🟢 Просветитель (Зелёный)
**Стремится к:** уровень 1-2, растёт постепенно
**Риск:** отдать ценный продукт почти бесплатно

Естественные продукты:
- Платформы для курсов и марафонов
- Wellness и оздоровительные приложения
- Трекеры привычек
- Платформы для мастермайндов
- Сообщества с платным доступом

Формат продажи: фиксированная цена установленная заранее, membership

Антипаттерн: взять личную ответственность за результат каждого участника

---

### 🟡 Мастер (Жёлтый)
**Стремится к:** уровень 1-2, может создать сложное если задача требует
**Риск:** перфекционизм, невидимость

Естественные продукты:
- Калькуляторы и практические инструменты
- Цифровые продукты (шаблоны, гайды, чек-листы)
- Генераторы документов
- Автоматизации для своего или чужого бизнеса
- Сложные системы если это нужно под задачу

Формат продажи: сарафанное радио, личный контакт, разовая оплата

Антипаттерн: бесконечная доработка без выпуска

---

### 🟠 Творец (Оранжевый)
**Стремится к:** уровень 1-2, всё живое и быстрое
**Риск:** не доводит до продажи, остывает к готовому продукту

Естественные продукты:
- Контент-инструменты (генераторы постов, контент-планов)
- Медиакиты и презентации
- Инструменты для коллабораций
- Боты для конкурсов и розыгрышей
- Живые события с технической поддержкой

Формат продажи: продавать в момент горения, предпродажи, живые события

Антипаттерн: долгосрочный продукт в одиночку без живого контакта

---

### 🔴 Спринтер (Красный)
**Стремится к:** уровень 1-2, быстро и с харизмой
**Риск:** нет удержания аудитории

Естественные продукты:
- Курсы и интенсивы с его лицом и голосом
- Боты для продаж и follow-up
- Системы лидогенерации
- Скрипты и воронки продаж

Формат продажи: агрессивный маркетинг, личный бренд, высокая скорость

Антипаттерн: тихий продукт без его присутствия, долгосрочное сопровождение

---

## Примечание
100 готовых решений курса (lesson.viberost.ru/solutions) покрывают уровни 1-2.
Техника создания одна — меняется предметная область.
Уровень 3 — уникален для каждого проекта, шаблонов нет.
Уровень 4 — в перспективе, следующий поток курса.
